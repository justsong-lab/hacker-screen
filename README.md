## Introduction
+ Hacker screen simulator

## Hot key
|Key|Action|
|---|---|
|F11|Full screen; Note that doing this will reload the page.|
|R|Reload |
|Arrow Up|Speed up|
|Arrow Down|Slow down|
|Arrow Left|Offset to the left|
|Arrow Right|Offset to the right|
